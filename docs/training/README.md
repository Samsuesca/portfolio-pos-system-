## Prompt para NotebookLM

```
Eres un asistente para crear presentaciones de capacitacion empresarial.

CONTEXTO:
- Negocio familiar de uniformes escolares en Colombia
- En transicion de informal a formal/digital
- Equipo: 5 jovenes familiares (18-22 anos, sin experiencia laboral formal)
- Sistema tecnologico recien implementado
- Modelo de compensacion: pago por dia + equity

OBJETIVO DE LA PRESENTACION:
1. Generar compromiso emocional con el proyecto (no es solo un trabajo)
2. Explicar claramente la propuesta de valor (equity vs salario tradicional)
3. Establecer expectativas claras (reglas no negociables)
4. Capacitar en el uso del sistema

TONO:
- Directo, sin rodeos
- Inspirador pero realista
- Familiar pero profesional
- Claro sobre que se espera y que se ofrece

AUDIENCIA:
- Jovenes sin experiencia empresarial
- Necesitan entender que el pago real es el proyecto, no el salario diario
- Deben sentirse co-constructores, no empleados

Basandote en los documentos adjuntos, crea una presentacion que logre estos objetivos.
```

## Elementos Clave a Enfatizar

### El Mensaje Central
> "No estas aqui para trabajar. Estas aqui para construir algo tuyo."

### La Diferencia
| Empleado | Co-Constructor |
|----------|----------------|
| Cobra y se va | Construye y crece |
| Sigue ordenes | Propone soluciones |
| Trabaja por horas | Trabaja por resultados |
| Al final: nada | Al final: participacion |

### La Apuesta
- Si funciona: Todos ganan mucho mas que un salario
- Si no funciona: Todos ganan habilidades y experiencia
- El unico que pierde: El que no lo intenta
