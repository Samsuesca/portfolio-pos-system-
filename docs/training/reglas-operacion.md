# Reglas de Operacion - El Codigo del Equipo

## Por Que Reglas

Un barco sin reglas es un barco que naufraga.

No son reglas para controlar. Son reglas para que todos sepamos como funciona el juego. Cuando todos conocen las reglas, nadie puede decir "no sabia".

**Estas reglas no son negociables.** Si no puedes seguirlas, este no es el lugar para ti.

---

## Las 7 Reglas Fundamentales

### Regla #1: TODO Se Registra

**El sistema es la unica fuente de verdad.**

| Accion | Donde se registra |
|--------|-------------------|
| Venta | Sistema - Modulo Ventas |
| Pedido | Sistema - Modulo Pedidos |
| Arreglo | Sistema - Modulo Arreglos |
| Gasto | Sistema - Modulo Contabilidad |
| Cambio/Devolucion | Sistema - Desde la venta original |
| Problema | Chat del equipo + Sistema si aplica |

**Si no esta en el sistema, no existe.**

No hay ventas "por fuera". No hay gastos "de bolsillo sin registrar". No hay "despues lo meto".

**Consecuencia de incumplimiento:**
- Primera vez: Advertencia seria + registro retroactivo
- Segunda vez: Perdida de bonificacion del periodo
- Tercera vez: Reconsideracion de tu participacion

---

### Regla #2: El Efectivo Es Sagrado

**El dinero en caja es del negocio. No es tuyo. No es de nadie.**

Principios:
- La caja se cuadra TODOS los dias antes de cerrar
- Diferencias se investigan y se explican
- No se "presta" plata de la caja para nada
- No se pagan cosas personales con plata del negocio
- El vuelto se calcula correctamente (el sistema ayuda)

**El sistema registra:**
- Saldo inicial del dia
- Cada venta y metodo de pago
- Cada gasto
- Saldo final esperado

Si el saldo fisico no coincide con el sistema, hay un problema. Y los problemas se resuelven, no se esconden.

**Consecuencia de incumplimiento:**
- Diferencia menor ($10,000 o menos): Explicacion + ajuste
- Diferencia repetida: Investigacion seria
- Faltante sin explicacion: Responsabilidad personal + posible salida

---

### Regla #3: El Cliente Es Prioridad (Pero No Siempre Tiene Razon)

**Principio:** Todo cliente merece respeto y atencion. Eso no significa que todo cliente tiene razon.

**Como tratar al cliente:**
- Saludo amable SIEMPRE
- Atencion completa (no celular mientras atiendes)
- Paciencia con preguntas
- Explicacion clara de productos y precios
- Agradecer la compra

**Cuando el cliente NO tiene razon:**
- Quiere devolucion fuera de politica → Escalar a supervisor
- Es grosero o agresivo → Mantener calma, escalar
- Pide descuentos no autorizados → "No estoy autorizado, dejeme consultar"
- Quiere credito sin historial → Seguir politica establecida

**Nunca:**
- Discutir con cliente
- Prometer algo que no puedes cumplir
- Hablar mal de otro cliente
- Dar informacion personal de otros clientes

**Consecuencia de mal trato a cliente:**
- Primera vez: Retroalimentacion privada
- Patron repetido: Reentrenamiento
- Cliente perdido por maltrato: Responsabilidad seria

---

### Regla #4: El Inventario Es Dinero En Forma de Tela

**Cada camisa, cada pantalon, cada prenda es dinero del negocio.**

**Responsabilidades:**
- Productos se manejan con cuidado
- Stock bajo se reporta INMEDIATAMENTE
- Perdidas o danos se reportan el mismo dia
- Inventario fisico debe coincidir con sistema

**El sistema muestra:**
- Stock actual por producto y talla
- Alertas de stock bajo
- Historial de movimientos

**Cuando hay discrepancia:**
- Se investiga: ¿Error de registro? ¿Robo? ¿Dano?
- Se documenta la causa
- Se ajusta en sistema con nota explicativa
- Si hay patron, se toman medidas

**Consecuencia:**
- Perdida por descuido: Responsabilidad compartida
- Perdida por negligencia grave: Responsabilidad personal
- Robo: Salida inmediata + acciones legales si aplica

---

### Regla #5: Los Datos Mandan, No Las Opiniones

**No tomamos decisiones porque "yo creo" o "a mi me parece".**

El sistema tiene la informacion:
- ¿Que producto se vende mas? → Reportes de ventas
- ¿Que talla pedir? → Historico de demanda
- ¿Como vamos este mes? → Dashboard
- ¿Podemos dar descuento? → Margen del producto

**Cuando alguien diga:**
- "Creo que deberiamos..." → "¿Que dicen los datos?"
- "A mi me parece que..." → "¿Podemos verificar en el sistema?"
- "Siempre se ha hecho asi..." → "¿Los datos muestran que funciona?"

**Esto no significa:**
- Que la intuicion no vale (Consuelo tiene anos de experiencia)
- Que no se escuchan ideas nuevas
- Que todo es rigido

**Significa:**
- Las decisiones importantes se respaldan con datos
- Si la intuicion contradice los datos, se investiga por que
- Aprendemos de los numeros, no de suposiciones

---

### Regla #6: Los Problemas Se Dicen, No Se Esconden

**Un problema pequeno escondido se vuelve problema grande.**

**Cultura de honestidad:**
- Si cometiste un error → Lo dices inmediatamente
- Si algo no funciona → Lo reportas
- Si no entiendes algo → Preguntas
- Si tienes una idea → La compartes
- Si no estas de acuerdo → Lo expresas respetuosamente

**Como reportar problemas:**
1. Describir que paso (sin excusas)
2. Describir el impacto
3. Proponer solucion si la tienes
4. Ejecutar o esperar instruccion

**Nunca:**
- Esconder un error esperando que nadie se de cuenta
- Culpar a otro sin evidencia
- Quedarse callado cuando algo esta mal
- Asumir que "alguien mas lo resolvera"

**Consecuencia:**
- Error reportado a tiempo: Se resuelve, se aprende, se sigue
- Error escondido y descubierto despues: Consecuencia doble por el error + por ocultarlo

---

### Regla #7: El Equipo Esta Sobre El Individuo

**Estamos construyendo algo juntos. El exito individual solo existe si hay exito colectivo.**

**Esto significa:**
- Ayudar al companero cuando esta abrumado
- Compartir conocimiento, no acapararlo
- Celebrar los logros del equipo, no solo los propios
- Aceptar feedback sin tomarlo personal
- Dar feedback honesto, no solo lo que quieren escuchar

**Nunca:**
- Sabotear a un companero para verse mejor
- Hablar mal de alguien a sus espaldas
- Negarse a ayudar porque "no me toca"
- Crear divisiones o "bandos"

**El exito de uno es exito de todos. El fracaso de uno afecta a todos.**

## Consecuencias Generales

### Nivel 1: Conversacion
- Primera falta menor
- Se habla en privado
- Se documenta internamente
- Oportunidad de corregir

### Nivel 2: Advertencia Formal
- Falta repetida o falta moderada
- Se documenta por escrito
- Plan de correccion claro
- Ultima oportunidad

### Nivel 3: Reconsideracion
- Patron de incumplimiento
- Falta grave
- Evaluacion de permanencia en el equipo
- Posible perdida de equity acumulado

### Nivel 4: Salida
- Incumplimiento grave o repetido
- Robo, fraude, o deshonestidad
- Salida del equipo
- Perdida total de equity no vesteado

## Horarios y Asistencia

### Temporada Alta
- Horario: 8:00 AM - 6:00 PM
- Almuerzo: 1 hora (coordinada para no cerrar)
- Dias: Lunes a Sabado
- Domingo: Segun demanda (se avisa con anticipacion)

### Puntualidad
- Llegar 5-10 minutos antes para preparar
- Si vas a llegar tarde: Avisar ANTES de la hora de entrada
- Ausencia: Avisar con minimo 12 horas de anticipacion (emergencias exceptuadas)

### Ausencias
- Emergencia real: Se entiende, se cubre
- Ausencia planeada: Con anticipacion para organizar cobertura
- Patron de ausencias: Se evalua compromiso

## Vestimenta y Presentacion

### Requerido
- Ropa limpia y presentable
- Uniforme del negocio cuando se proporcione
- Aspecto aseado

### Opcional pero Recomendado
- Colores neutros o del negocio
- Calzado comodo pero presentable

### No Permitido
- Ropa sucia o rota
- Chanclas o sandalias de playa
- Gorras (dentro del local)
- Ropa con mensajes inapropiados

---

## Uso de Celular

### Permitido
- Consultas rapidas relacionadas al trabajo
- Comunicacion del chat del equipo
- Emergencias familiares

### No Permitido
- Redes sociales mientras atiendes
- Llamadas personales largas en horario
- Videos o juegos

**Principio:** Si hay cliente presente, el celular no existe.

---

## Seguridad

### Apertura
- Verificar antes de abrir
- Contar caja inicial
- Verificar que todo funciona

### Cierre
- Verificar que no hay clientes
- Cuadrar caja y registrar
- Asegurar inventario valioso
- Cerrar con llave, verificar

### Dinero
- No contar dinero a vista de todos
- Depositos frecuentes si hay mucho efectivo
- No comentar sobre cuanto hay en caja

### Emergencias
- Conocer salidas de emergencia
- Numero de contacto de emergencias visible
- En caso de robo: Colaborar, no ser heroe, reportar inmediato
